# slugify-text

Convert strings to URL-friendly slugs.

## Why

Turn messy titles, names, and labels into clean, shareable URL slugs without thinking about edge cases.

## Features

- Lowercases and trims whitespace.
- Replaces spaces and special characters with hyphens.
- Tiny dependency-free utility.

## Quick start

```js
const slugify = require('slugify-text');

slugify('Hello World!'); // "hello-world"
```

## Install

```bash
npm install slugify-text
```
